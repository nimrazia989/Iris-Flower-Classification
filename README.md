# 🌸 Iris Flower Classification (Python ML Project)

This is a beginner-friendly Machine Learning project to classify iris flowers into three species setosa, versicolor, or virginica using the famous Iris dataset.

---

## 🚀 Features
- Load and explore the Iris dataset
- Split data into training and testing sets
- Scale features using StandardScaler
- Train a Random Forest Classifier
- Predict the class of new samples
- Evaluate model performance (accuracy and classification report)

---

## 🛠️ Technologies Used
- Python
- scikit-learn
- pandas (optional for data exploration)
- numpy
